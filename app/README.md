# Project README

## Overview

This project is a Next.js application that utilizes Tailwind CSS for styling and Google Fonts for typography. The application is structured with a root layout and a home page, and it includes global styles for consistent theming.

## File Structure

The repository contains the following key files:

1. `app/layout.tsx`
2. `app/page.tsx`
3. `app/globals.css`

### 1. `app/layout.tsx`

This file defines the root layout of the application. It imports necessary modules and sets up global metadata and font styles.

#### Key Entities:

- **Metadata**: Contains the title and description of the application.
- **Geist and Geist_Mono**: Google Fonts used for the application.
- **RootLayout Component**: Wraps the entire application, applying global styles and fonts.

#### Code Breakdown:

```tsx
import type { Metadata } from "next";
import { Geist, Geist_Mono } from "next/font/google";
import "./globals.css";

const geistSans = Geist({
  variable: "--font-geist-sans",
  subsets: ["latin"],
});

const geistMono = Geist_Mono({
  variable: "--font-geist-mono",
  subsets: ["latin"],
});

export const metadata: Metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html lang="en">
      <body
        className={`${geistSans.variable} ${geistMono.variable} antialiased`}
      >
        {children}
      </body>
    </html>
  );
}
```

### 3. `app/globals.css`

This file contains global CSS styles for the application, including Tailwind CSS imports and custom properties for theming.

#### Key Entities:

- **Tailwind CSS**: Base, components, and utilities are imported for styling.
- **Custom Properties**: Defines background and foreground colors for light and dark modes.
- **Global Styles**: Applies the custom properties to the body element.

#### Code Breakdown:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

:root {
  --background: #ffffff;
  --foreground: #171717;
}

@media (prefers-color-scheme: dark) {
  :root {
    --background: #0a0a0a;
    --foreground: #ededed;
  }
}

body {
  color: var(--foreground);
  background: var(--background);
  font-family: Arial, Helvetica, sans-serif;
}
```

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository**:
   ```sh
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Install dependencies**:
   ```sh
   npm install
   ```

3. **Run the development server**:
   ```sh
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to see the application in action.

This README provides a comprehensive overview of the project's structure and key components. By following the instructions, you can easily set up and run the application locally. For further customization, refer to the code and modify it as needed.
